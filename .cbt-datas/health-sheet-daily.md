# 健康管理シート（日間）＋CBT — 記入テンプレート

心療内科・精神科に持参する「生活記録表」を、`logs/` とCBTセルフケア機能に接続できるようMarkdown化した**1日分の記入テンプレート**です。紙のシートのアイコン項目（起床・睡眠・体調・心理・生活状況・服薬・その他）に、日間はCBTの記録欄（気分スコア・思考記録・行動活性化・感謝日記）を足してある。

## 使い方

- 下の「テンプレート」ブロックをコピーして日付を入れ、書ける欄だけ埋める。全部埋めなくていい。
- 見出しは [logs/README.md](../logs/README.md) のフォーマットに合わせてあるので、そのまま `logs/YYYY-MM-DD.md` として保存すれば、Claude連携ブリッジ経由でMisskey Botにも引き継がれ、[気分推移Artifact](./mood-artifact.md) の `気分: N/10` も自動で拾える。
- 会話で埋めたい場合は [daily-checkin.md](./daily-checkin.md) の聞き方に沿って100(モモ)が一緒に記入を手伝う。CBTの各欄は [thought-record.md](./thought-record.md) / [behavioral-activation.md](./behavioral-activation.md) / [gratitude.md](./gratitude.md) を参照。
- 数値化を望まない欄は空欄・自由記述で構わない。**気分の数値を推測・捏造しない。**

## アイコン対応（紙シート → 見出し）

| 紙シート | 記入欄 |
| --- | --- |
| 👁‍🗨 目覚め・起床時刻 / 💤 昨夜の眠りの質・入眠時刻 | ## 睡眠・生活習慣 |
| 👤 身体的な体調 | ## 体調・気分（身体） |
| ❤️ 心理的な状態 | ## 体調・気分（心理）＋気分: N/10 |
| 💼 その他の生活状況 | ## 睡眠・生活習慣 / ## 取り組んだタスク |
| 💊 その他の内服薬 / 服薬（朝・日中・食後・夜・発作時⚡） | ## 服薬 |
| 💭 その他 | 各見出しの自由記述 |

## 記入例（イメージ）

```markdown
# 2026-07-09（木）

## 服薬

- 朝🌄: [x] 済 ／ 夜🌙: [x] 済

## 体調・気分

気分: 6/10

- ❤️ 心理的な状態: 朝は「すべて無駄」感で無気力。行動したら午後は回復。
- 👤 身体的な体調: エネルギー 7/10。夕方は十分動けた。

## 睡眠・生活習慣

- 👁‍🗨 起床時刻: 8:00 ／ 💤 入眠 1:30・眠りは浅め(2/5)
- 💼 その他: 昼に散歩10分

## 行動活性化

- 計画: 小さな作業を先に1つ着手 → 実施でき、無気力が軽くなった

## 感謝日記

1. 自力で気分を持ち上げられた
2. 作業が捗った
```
