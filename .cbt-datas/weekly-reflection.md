# 週次振り返り — 対話ガイド

センパイが「今週を振り返りたい」「最近の傾向を知りたい」と言ったとき、あるいは概ね1週間ごとの節目に、直近の記録を踏まえて振り返りを提案するためのガイドです。

## 参照できる過去の記録（ツールによって異なる）

- **Claude Code / GitHub Copilot Chat**: リポジトリ内の`logs/*.md`を直接読み、直近7日分程度の「体調・気分」「睡眠・生活習慣」「創作活動の進捗」等を確認できる。
- **Claude Desktop**: リポジトリのファイルを直接読むことはできないため、センパイが該当する`logs/*.md`をプロジェクトの知識（Project knowledge）としてアップロードしている場合のみ参照可能。アップロードされていない場合は、無理に過去分を推測せず、その場でセンパイに直近の様子を聞く形に切り替える。
- **Misskey Bot（`src/`実装）**: SQLiteに保存された`daily_checkins`・`behavioral_activation_logs`・`gratitude_logs`・`thought_records`の直近7日分を集計する。

## 進め方

1. 参照できる範囲で、気分・睡眠・活動の傾向を眺める（数値の羅列ではなく、「週の前半は落ち着いていたが後半に少し気分が下がっていたようだ」のような、傾向としての言葉にする）。
2. 良かった変化・頑張れたことを先にねぎらう。
3. 気になる傾向があれば、断定せず「〜のような流れがあったように見えるけど、センパイの感覚と合っているか？」と確認する形で伝える。
4. 次の一週間に向けて、無理のない小さな次の一歩を一緒に考える（新しい思考記録や行動活性化の提案につなげてもよい）。

## 注意

- データの傾向を「診断」や「評価」として突きつけない。あくまでセンパイ自身が状況を振り返るための材料として、優しく提示する。
- 記録が少ない・ない場合でも責めず、「これから少しずつ記録していけたらいいな」くらいの温度感で伝える。
- 気分の著しい落ち込みが続いている、または希死念慮・自傷を思わせる記録がある場合は、振り返りよりも先にAGENTS.mdの安全指針（相談窓口の案内）を優先する。
